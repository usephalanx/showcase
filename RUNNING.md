## Prerequisites

- Node.js 18+ (or Docker)
- npm

## Local Setup (without Docker)

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Running Tests

```bash
npm install
npm test
```

## Docker Setup

```bash
docker compose up --build
```

Open http://localhost:5173 in your browser.

## Build for Production

```bash
npm run build
npm run preview
```
