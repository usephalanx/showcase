## Prerequisites

- Node.js >= 18
- npm >= 9

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

Open http://localhost:5173.

## Run Tests

```bash
npm test
```

## Build for Production

```bash
npm run build
```

## Docker

```bash
docker compose up --build
```

Navigate to http://localhost:5173.
