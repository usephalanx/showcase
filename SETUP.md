# Setup

This project uses npm for dependency management. The following files are
generated by tooling and must **not** be hand-written:

- `node_modules/`
- `package-lock.json`
- `dist/`

## Install dependencies

```bash
npm install
```

## Run development server

```bash
npm run dev
```

## Run tests

```bash
npm test
```

## Build for production

```bash
npm run build
```
