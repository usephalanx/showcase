# Frontend Setup

The following files are **not** committed and must be generated by running the project's toolchain:

- `frontend/node_modules/` — installed packages
- `frontend/package-lock.json` — lockfile (generated by npm)

## Steps

```bash
# 1. Install Node.js dependencies
cd frontend
npm install

# 2. Start the dev server (proxies /api to localhost:8000)
npm run dev

# 3. Build for production
npm run build
```

## Running Tests

Tests are Python-based and validate file structure without needing Node.js:

```bash
# From the project root
pip install pytest
pytest tests/test_frontend_setup.py -v
```
