# Setup

## Prerequisites

- Node.js >= 18
- npm >= 9

## Install Dependencies

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Run Tests

```bash
npm run test
```

## Notes

Do **not** commit `node_modules/`, `package-lock.json`, or `dist/`.
These are generated by the toolchain:

```bash
# After cloning, regenerate lock file and install:
npm install
```
