# Setup Instructions

The following lock files and generated directories are **not** committed
to version control. Run these commands after cloning to generate them.

## Frontend

```bash
cd frontend
npm install          # generates package-lock.json and node_modules/
npm run build        # generates dist/ output directory
```

## Python Backend (existing)

```bash
pip install -r requirements.txt   # if a requirements file exists
```

## Files that must NOT be hand-written

- `frontend/package-lock.json` — generated by `npm install`
- `frontend/node_modules/` — generated by `npm install`
- `frontend/dist/` — generated by `npm run build`
- `__pycache__/` — Python bytecode cache
- `.venv/` — Python virtual environment
