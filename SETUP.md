# Setup Instructions

## Prerequisites

- Python 3.10+
- pip

## Install Dependencies

```bash
pip install fastapi sqlalchemy pydantic python-slugify pytest
```

## Run Tests

```bash
cd /path/to/project
python -m pytest tests/ -v
```

## Notes

- Do NOT commit `__pycache__/`, `.venv/`, `*.egg-info/`, `poetry.lock`, or `Pipfile.lock`.
- These are generated by your toolchain and should be in `.gitignore`.
