# Setup Instructions

## Frontend Dependencies

The following files are generated by package managers and must NOT be hand-written.
Run these commands to generate them:

```bash
cd frontend
npm install
```

This will generate:
- `frontend/node_modules/` — installed dependencies
- `frontend/package-lock.json` — exact dependency lock file

## Building the Frontend

```bash
cd frontend
npm run build
```

This generates the `frontend/dist/` directory with optimized production assets.

## Running Tests

```bash
# From project root
pip install pytest
pytest tests/
```
