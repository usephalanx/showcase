# Setup Instructions

## Install dependencies

```bash
npm install
```

## Run development server

```bash
npm run dev
```

## Run tests

```bash
npm test
```

## Build for production

```bash
npm run build
```

## Notes

Do **not** hand-edit or commit the following generated files:
- `node_modules/`
- `package-lock.json`
- `dist/`

These are produced by running the commands above.
