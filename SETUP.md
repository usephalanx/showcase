# Setup Instructions

## Prerequisites

- Python 3.11+
- pip

## Backend Setup

```bash
# Create virtual environment
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate

# Install dependencies
pip install -r backend/requirements.txt

# Install test dependencies
pip install pytest python-slugify

# Run database initialization (creates tables)
cd backend
python -c "from database import init_db; init_db()"

# Generate initial Alembic migration
alembic revision --autogenerate -m "Initial migration"

# Apply migrations
alembic upgrade head
```

## Running Tests

```bash
# From the project root
python -m pytest tests/ -v

# Run planning validation
python scripts/validate_planning.py
```

## Auto-generated Files (Do NOT commit)

The following are generated by tooling and should not be hand-written:

- `*.egg-info/`
- `__pycache__/`
- `.venv/`
- `*.db` (SQLite database files)
- `backend/alembic/versions/*.py` (generated by `alembic revision --autogenerate`)
