# Setup Instructions

## Frontend

### Install Dependencies

```bash
cd frontend
npm install
```

### Run Tests

```bash
cd frontend
npm test
```

### Type Check

```bash
cd frontend
npm run typecheck
```

### Development Server

```bash
cd frontend
npm run dev
```

### Build

```bash
cd frontend
npm run build
```

## Lock Files

The following files are generated by tooling and should NOT be hand-written:

- `frontend/package-lock.json` — generated by `npm install`
- `frontend/node_modules/` — generated by `npm install`
- `frontend/dist/` — generated by `npm run build`

Run `npm install` in the `frontend/` directory to generate the lock file.
