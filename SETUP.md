# Setup Instructions

## Prerequisites

- Node.js 18+
- npm 9+ (or pnpm / yarn)

## Install Dependencies

```bash
npm install
```

## Development Server

```bash
npm run dev
```

## Run Tests

```bash
npm run test
```

## Build for Production

```bash
npm run build
```

## Notes

- Lock files (`package-lock.json`, `yarn.lock`, `pnpm-lock.yaml`) are generated by your package manager and should not be hand-edited.
- The `node_modules/` directory is generated by `npm install` and must not be committed.
- Build output (`dist/`) is generated by `npm run build` and must not be committed.
