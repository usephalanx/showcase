# Setup Instructions

## Prerequisites

- Node.js >= 18
- npm >= 9
- Python >= 3.10 (for backend and tests)

## Frontend Setup

```bash
# Install JavaScript dependencies (generates package-lock.json and node_modules/)
npm install

# Start the development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Running Tests

```bash
# Install Python test dependencies
pip install pytest

# Run frontend scaffolding tests
pytest tests/test_frontend_setup.py -v
```

## Notes

The following files are generated by tooling and should NOT be committed manually:

- `package-lock.json` — generated by `npm install`
- `node_modules/` — generated by `npm install`
- `dist/` — generated by `npm run build`
