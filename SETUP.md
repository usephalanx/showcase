# Setup Instructions

## Local Development (without Docker)

```bash
# Create and activate a virtual environment
python3 -m venv .venv
source .venv/bin/activate   # Linux/macOS
# .venv\Scripts\activate    # Windows

# Install dependencies
pip install -r requirements.txt

# Run the application
uvicorn app.main:app --reload

# Run tests
pytest tests/
```

## Docker

```bash
# Build and start
docker-compose up --build

# Or build manually
docker build -t hello-world-api .
docker run -p 8000:8000 hello-world-api
```

## Verify

```bash
curl http://localhost:8000/       # {"status": "ok"}
curl http://localhost:8000/hello   # {"message": "hello-world"}
```

## Auto-generated files (do NOT commit)

The following are generated by tooling and should not be hand-written:

- `__pycache__/`, `*.pyc`
- `.venv/`, `*.egg-info/`
- `poetry.lock`, `Pipfile.lock`
